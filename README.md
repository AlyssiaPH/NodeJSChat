# NodeJSChat

## Présentation
Ce dépôt contient une api de Chat  avec un système d'authentification. L'utilisateur aura accès a cette application uniquement après s'être authentifier.

### pré-requis:
- Node.js ( [Node.js](https://nodejs.org/en/) )
dépendances : 
- mongoose 
- express 
- axioss
- webjsontoken
exécuter la commande `npm install` pour installer le package.

### guide
1. Ouvrir une invite de commande à la racine du projet
2. S'assurer d'avoir les dépendance npm installées 
3. Exécuter la commande `npm dev.serve`
 
Puis :
4. Ouvrir un navigateur à l'adresse donné
